Git y GitHub

¿Qué es Git?

Git es un sistema de control de versiones distribuido. Cada desarrollador trabaja con una copia completa del historial del proyecto de forma local, sin depender de un servidor central, lo que permite mayor autonomía y seguridad.

¿Qué es GitHub?

GitHub es una plataforma en línea que permite almacenar repositorios Git en la nube, facilitando la colaboración entre desarrolladores, el seguimiento de cambios y el control de versiones. Fue adquirida por Microsoft en 2018.

Principales Diferencias

Git GitHub
Es un sistema de control de versiones local. No requiere conexión a internet. Guarda y administra versiones del código. Es una plataforma basada en Git, en línea. Requiere conexión a internet para colaborar y compartir proyectos.
